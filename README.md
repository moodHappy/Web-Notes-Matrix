# 🕸️ Web Matrix 枢纽

**Web Matrix** 是一个为语法学习者打造的“高密度信息矩阵”。它将杂乱的英语语法笔记转化为结构化的矩阵数据，通过 GitHub 自动化工作流与精简的前端架构，实现了一个可随身携带、具备全栈即时编辑能力的语法实验室。

---

## 🛠 矩阵架构 (Architecture)

Web Matrix 不仅仅是静态页面，它是一个高度集成的 Web 应用：

* **全自动索引系统**：通过 `build_calendar.py` 扫描所有语法解构记录，自动生成支持日期导航的日历枢纽。
* **云端矩阵同步**：内置了 GitHub API 深度整合，无需后台数据库，直接将笔记推送到仓库，实现“所见即所得”的云端持久化。
* **移动端优先设计**：极致紧凑的 CSS 排版，支持手势操作（双击或双指触摸触发编辑器），让碎片化时间的语法解构变得轻松。
* **静默编辑架构**：每个解构页面都具备自修复与自同步能力。您在页面上对 Markdown 的任何修改，都会触发云端的静默更新。

---

## 🚀 核心功能

### 1. 语法解构实验室 (Deconstruct Room)
在“矩阵录入”页面，您可以将零散的英语原句与深度 Markdown 分析推送到仓库。系统自动生成归档，并按日期自动排列至主枢纽。

### 2. 交互式历史枢纽 (Calendar Hub)
日历视图实时显示您的学习进度。绿色圆点标记着每一份被解构的语法记录，点击即可快速回顾。

### 3. 动态 Markdown 渲染
基于 `marked.js` 的动态渲染，确保您可以随时通过手机端双击界面，利用全屏幕编辑器进行深度修改，即刻保存至 GitHub。

## ⚙️ 快速启用 (Deployment)

1.  **Fork 本仓库**。
2.  在 GitHub 仓库中，进入 `Settings` -> `Actions` -> `General`，确保已开启 `Read and write permissions`。
3.  项目会自动扫描 `docs/` 下的历史文件并编译 `index.html`。
4.  **配置访问权限**：
    * 在矩阵枢纽首页点击右上角 **⚙️**。
    * 填入您的 **GitHub Personal Access Token**（该 Token 仅用于您个人的仓库读写，保存在浏览器 `localStorage` 中，绝对安全）。

---

## 🤖 自动化工作流 (Workflow)

本项目包含一套闭环的自动化机制：

* **推送监听**：每当您通过网页端或 IDE 推送新的 `.html` 笔记时，GitHub Actions 会自动触发 `Sync Web Notes Calendar` 工作流，重新构建日历枢纽，确保矩阵永远处于最新状态。
